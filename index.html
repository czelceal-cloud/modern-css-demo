04 Container Queries

This grid reflows based on its container's width — not the viewport. Resize the browser to see it adapt.

📦 Container

Uses container-type: inline-size so children query its width, not the viewport.

🔄 Reflows

When the container shrinks below 600px, two columns become one.

📐 Responsive

Container queries decouple sizing from viewport breakpoints — perfect for reusable components.

🧩 Composable

Drop this anywhere; it adapts to its parent, not the screen size.
